# accounts-manage-and-hold
This project is a simple banking management system implemented using C++. It allows users to create accounts, deposit money, withdraw funds, and view account details. Data is persisted using file handling.  The goal of this project is to understand system design, structured data, and basic financial transaction workflows
Features
- Create new account with auto-generated account number
- Deposit money
- Withdraw money with balance validation
- Search accounts
- Persistent storage using files
- Menu-driven interface
